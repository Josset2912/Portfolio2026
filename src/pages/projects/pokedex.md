Desarrollé una Pokédex web interactiva utilizando JavaScript, HTML y CSS, que permite consultar y visualizar información detallada de distintos Pokémon. Implementé consumo de datos mediante una API pública (PokéAPI), incorporando diseño responsive y una interfaz intuitiva orientada a la experiencia del usuario.
